记录第一次搭建hexo

发表于 2016-11-22 | | 阅读次数

即使扒了很多大佬的搭建教程，还是踩了很多坑，所以打算记录下来自己的搭建过程。本文主要介绍win10下hexo3.1.1搭建，并发布到GitHub上。

环境搭建

 安装nodejs（必须）

  • 作用：用来生成静态页面的

  • 方法：到node.js官网下载并安装。我是用的是win10下v6.9.1版本。

     安装git（必须）

  • 作用：把本地的hexo内容提交到github上去。

  • 方法：进入git官方下载即可，速度较慢。

     申请GitHub账号（必须）

  • 作用：是用来做博客的远程创库、域名、服务器之类的。
  • 方法：到github官网自行注册，如果想深入学习请看pro git教程
